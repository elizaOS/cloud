# Credits & Billing

<div className="status-badge status-stable">Stable</div>

Manage your credits, top up, and view usage.

## Get Credit Balance

<div className="api-endpoint">
  <span className="method-badge method-badge-get">GET</span>
  <span className="path">/api/v1/credits/summary</span>
</div>

Get your current credit balance.

### Response

```json
{
  "balance": 50.00,
  "currency": "USD",
  "usage": {
    "today": 2.50,
    "thisMonth": 45.00
  }
}
```

## Top Up Credits (x402)

<div className="api-endpoint">
  <span className="method-badge method-badge-post">POST</span>
  <span className="path">/api/v1/credits/topup</span>
</div>

Add credits using x402 payment.

### Request

```json
{
  "amount": 100.00,
  "paymentMethod": "x402"
}
```

### Response

```json
{
  "success": true,
  "newBalance": 150.00,
  "transaction": {
    "id": "txn_abc123",
    "amount": 100.00,
    "createdAt": "2024-01-15T10:30:00Z"
  }
}
```

## Get Transactions

<div className="api-endpoint">
  <span className="method-badge method-badge-get">GET</span>
  <span className="path">/api/v1/credits/transactions</span>
</div>

Get transaction history.

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `limit` | integer | Results per page (default: 20) |
| `offset` | integer | Pagination offset |
| `type` | string | `topup`, `usage`, `refund` |

### Response

```json
{
  "transactions": [
    {
      "id": "txn_abc123",
      "type": "usage",
      "amount": -0.05,
      "description": "Chat completion (gpt-4o)",
      "createdAt": "2024-01-15T10:30:00Z"
    }
  ],
  "total": 150
}
```

## Auto Top-Up Settings

<div className="api-endpoint">
  <span className="method-badge method-badge-get">GET</span>
  <span className="path">/api/auto-top-up/settings</span>
</div>

Get auto top-up configuration.

<div className="api-endpoint">
  <span className="method-badge method-badge-put">PUT</span>
  <span className="path">/api/auto-top-up/settings</span>
</div>

Configure auto top-up.

### Request

```json
{
  "enabled": true,
  "threshold": 10.00,
  "amount": 50.00
}
```

<Callout type="info">
Auto top-up automatically adds credits when your balance falls below the threshold.
</Callout>
