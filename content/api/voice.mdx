# Voice & TTS

<div className="status-badge status-stable">Stable</div>

Text-to-speech generation and voice cloning capabilities powered by ElevenLabs.

## Text to Speech

<div className="api-endpoint">
  <span className="method-badge method-badge-post">POST</span>
  <span className="path">/api/elevenlabs/tts</span>
</div>

Convert text to speech audio using premium AI voices.

<Tabs items={['cURL', 'JavaScript', 'Python']}>
  <Tabs.Tab>
```bash
curl -X POST "https://elizacloud.ai/api/elevenlabs/tts" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello, this is a test of the text to speech system.",
    "voiceId": "21m00Tcm4TlvDq8ikWAM",
    "modelId": "eleven_multilingual_v2"
  }' \
  --output speech.mp3
```
  </Tabs.Tab>
  <Tabs.Tab>

  </Tabs.Tab>

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `text` | string | ✓ | Text to convert to speech (max 5000 chars) |
| `voiceId` | string | ✓ | Voice ID to use (see List Voices) |
| `modelId` | string | | Model ID. Default: `eleven_multilingual_v2` |
| `stability` | number | | Voice stability (0-1). Default: 0.5 |
| `similarity_boost` | number | | Voice similarity (0-1). Default: 0.75 |

### Available Models

| Model | Languages | Quality | Speed | Credits |
|-------|-----------|---------|-------|---------|
| `eleven_multilingual_v2` | 29 | Highest | Medium | 3 |
| `eleven_turbo_v2_5` | 32 | High | Fast | 2 |
| `eleven_monolingual_v1` | English | Good | Fastest | 1 |

### Response

Returns audio data as `audio/mpeg` stream. The `Content-Length` header indicates file size.

## Speech to Text

<div className="api-endpoint">
  <span className="method-badge method-badge-post">POST</span>
  <span className="path">/api/elevenlabs/stt</span>
</div>

Transcribe audio to text.

### Request

Upload audio as `multipart/form-data`:

```bash
curl -X POST "https://elizacloud.ai/api/elevenlabs/stt" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "audio=@recording.mp3"
```

### Response

```json
{
  "text": "Hello, this is a transcription test.",
  "language": "en",
  "confidence": 0.95
}
```

## Clone Voice

<div className="api-endpoint">
  <span className="method-badge method-badge-post">POST</span>
  <span className="path">/api/elevenlabs/voices/clone</span>
</div>

Create a voice clone from audio samples.

### Request

Upload audio samples as `multipart/form-data`:

```bash
curl -X POST "https://elizacloud.ai/api/elevenlabs/voices/clone" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "name=My Voice" \
  -F "files=@sample1.mp3" \
  -F "files=@sample2.mp3"
```

### Response

```json
{
  "id": "voice_xyz789",
  "name": "My Voice",
  "status": "processing"
}
```

<Callout type="info">
  **Voice Cloning Tips:**
  - Provide 1-5 minutes of clear audio for best results
  - Use high-quality recordings with minimal background noise
  - Speaking clearly and at a natural pace produces better clones
  - Multiple samples in different contexts improve voice quality
</Callout>

## Verify Voice

<div className="api-endpoint">
  <span className="method-badge method-badge-get">GET</span>
  <span className="path">/api/elevenlabs/voices/verify/{'{id}'}</span>
</div>

Check voice cloning status. Cloning typically takes 30-60 seconds.

### Response

```json
{
  "id": "voice_xyz789",
  "status": "ready",
  "quality": 0.92
}
```

### Status Values

| Status | Description |
|--------|-------------|
| `processing` | Voice clone is being generated |
| `ready` | Voice is ready to use |
| `failed` | Cloning failed (check audio quality) |

## Error Handling

| Code | Error | Solution |
|------|-------|----------|
| 400 | Invalid voice ID | Use a valid voice from List Voices |
| 400 | Text too long | Split text into chunks under 5000 chars |
| 402 | Insufficient credits | Add credits to your account |
| 404 | Voice not found | Voice may have been deleted |
| 429 | Rate limited | Wait and retry |
