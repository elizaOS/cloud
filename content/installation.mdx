# Installation

Set up your development environment for elizaOS Cloud integration.

## SDK Installation

Install the official elizaOS packages:

<Tabs items={['npm', 'pnpm', 'bun', 'yarn']}>
  <Tabs.Tab>
```bash
npm install @elizaos/core @elizaos/plugin-elizacloud
```
  </Tabs.Tab>
  <Tabs.Tab>
```bash
pnpm add @elizaos/core @elizaos/plugin-elizacloud
```
  </Tabs.Tab>
  <Tabs.Tab>
```bash
bun add @elizaos/core @elizaos/plugin-elizacloud
```
  </Tabs.Tab>
  <Tabs.Tab>
```bash
yarn add @elizaos/core @elizaos/plugin-elizacloud
```
  </Tabs.Tab>
</Tabs>

## CLI Installation

Install the elizaOS CLI for deployment and management:

```bash
npm install -g @elizaos/cli
```

Verify the installation:

```bash
elizaos --version
```

## Requirements

| Requirement | Version |
|-------------|---------|
| Node.js | 18.0+ |
| npm/pnpm/yarn/bun | Latest |
| TypeScript | 5.0+ (recommended) |

## Project Setup

<Steps>
### Initialize a New Project

```bash
elizaos init my-agent
cd my-agent
```

### Install Dependencies

```bash
npm install
```

### Configure Environment

Create a `.env` file:

```bash
# elizaOS Cloud Configuration
ELIZA_API_KEY=your_api_key_here
ELIZA_API_URL=https://elizacloud.ai/api/v1

# Optional: Model Provider Keys (if using direct providers)
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
```

### Start Development

```bash
npm run dev
```
</Steps>

## TypeScript Configuration

For TypeScript projects, ensure your `tsconfig.json` includes:

```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "esModuleInterop": true,
    "strict": true,
    "skipLibCheck": true
  }
}
```
