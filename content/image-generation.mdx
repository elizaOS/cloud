# Image Generation

Generate high-quality images from text prompts using state-of-the-art AI models.

<div className="status-badge status-stable">Stable</div>

## Overview

elizaOS Cloud provides image generation through Fal.ai, supporting multiple models:

- **FLUX Schnell**: Fast, high-quality generation
- **FLUX Pro**: Premium quality with more detail
- **Stable Diffusion XL**: Versatile, customizable
- **Stable Diffusion 3**: Latest SD architecture

## Quick Start

### Dashboard

Navigate to [Dashboard → Images](/dashboard/image) for the visual interface.

## Available Models

| Model                        | Speed   | Quality | Best For                           |
| ---------------------------- | ------- | ------- | ---------------------------------- |
| `fal-ai/flux/schnell`        | ⚡ Fast | High    | Quick iterations, prototyping      |
| `fal-ai/flux/pro`            | Medium  | Highest | Final production images            |
| `fal-ai/flux/dev`            | Medium  | High    | Development and testing            |
| `fal-ai/stable-diffusion-v3` | Medium  | High    | Versatile, good for most use cases |

## Request Parameters

| Parameter         | Type    | Required | Description                                   |
| ----------------- | ------- | -------- | --------------------------------------------- |
| `prompt`          | string  | ✓        | Text description of the image                 |
| `model`           | string  |          | Model to use (default: `fal-ai/flux/schnell`) |
| `size`            | string  |          | Output size (see sizes below)                 |
| `num_images`      | integer |          | Number of images (1-4, default: 1)            |
| `seed`            | integer |          | Random seed for reproducibility               |
| `guidance_scale`  | number  |          | How closely to follow the prompt (1-20)       |
| `negative_prompt` | string  |          | What to avoid in the image                    |

### Available Sizes

| Size             | Dimensions | Aspect Ratio |
| ---------------- | ---------- | ------------ |
| `square`         | 512x512    | 1:1          |
| `square_hd`      | 1024x1024  | 1:1          |
| `portrait_4_3`   | 768x1024   | 3:4          |
| `portrait_16_9`  | 576x1024   | 9:16         |
| `landscape_4_3`  | 1024x768   | 4:3          |
| `landscape_16_9` | 1024x576   | 16:9         |

## Prompt Engineering

### Basic Structure

```
[Subject] [Style] [Details] [Lighting] [Mood]
```

### Example Prompts

**Photorealistic:**

```
Portrait of a woman with silver hair, professional photography,
soft studio lighting, shallow depth of field, 8k quality
```

**Artistic:**

```
Abstract painting of ocean waves, impressionist style,
vibrant blues and teals, textured brushstrokes
```

**Product:**

```
Minimalist product shot of a smartwatch on white marble,
clean lighting, commercial photography, high detail
```

### Negative Prompts

Use negative prompts to exclude unwanted elements:

```json
{
  "prompt": "Professional headshot of a businessman",
  "negative_prompt": "blurry, distorted, low quality, cartoon"
}
```

## Advanced Usage

### Batch Generation

Generate multiple variations:

```javascript
const response = await fetch("https://elizacloud.ai/api/v1/generate-image", {
  method: "POST",
  headers: {
    Authorization: "Bearer YOUR_API_KEY",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    prompt: "A serene mountain landscape",
    num_images: 4,
    seed: 42, // Same seed for consistency
  }),
});
```

### Reproducible Results

Use a seed for consistent outputs:

```json
{
  "prompt": "A red sports car",
  "seed": 12345,
  "guidance_scale": 7.5
}
```

<Callout type="info">
  Using the same prompt, seed, and parameters will produce identical results.
</Callout>

## Error Handling

### Common Errors

| Error                  | Cause                          | Solution                    |
| ---------------------- | ------------------------------ | --------------------------- |
| `CONTENT_POLICY`       | Prompt violates content policy | Modify prompt to comply     |
| `RATE_LIMITED`         | Too many requests              | Wait and retry with backoff |
| `INSUFFICIENT_CREDITS` | Not enough credits             | Top up your account         |
| `INVALID_MODEL`        | Model not available            | Use a supported model       |

### Error Response

```json
{
  "error": {
    "code": "CONTENT_POLICY",
    "message": "The prompt violates our content policy",
    "details": "Please modify your prompt"
  }
}
```

## Best Practices

- **Be Specific** — Detailed prompts with style, lighting, and mood produce better results
- **Use Negative Prompts** — Exclude unwanted elements like "blurry, distorted, low quality"
- **Iterate** — Generate multiple variations with `num_images: 4` and refine your prompt
- **Consistent Seeds** — Use the same `seed` value to reproduce and iterate on good results
