# Eliza Cloud v2.0 Released

We're thrilled to announce the release of Eliza Cloud v2.0, our biggest update yet!

## What's New

### Performance Improvements

- **50% faster response times** across all models
- Reduced cold start latency
- Optimized memory usage

### New Features

#### Multi-Agent Orchestration

Create workflows that involve multiple agents working together:

```javascript
const workflow = {
  agents: ["researcher", "writer", "editor"],
  flow: "sequential",
  handoff: "automatic",
};
```

#### Enhanced Analytics Dashboard

- Real-time usage metrics
- Conversation quality scores
- Cost optimization insights

#### Improved API

- Streaming responses
- Batch processing
- Webhook support

### Developer Experience

- New CLI tools
- Better error messages
- Comprehensive documentation

## Migration Guide

Upgrading from v1.x is straightforward. Most changes are backward compatible, but we recommend reviewing the [migration guide](/docs/migration) for detailed instructions.

## Thank You

This release wouldn't be possible without our amazing community. Thank you for your feedback, bug reports, and feature requests!
