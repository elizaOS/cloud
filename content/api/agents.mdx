# Agents

<div className="status-badge status-stable">Stable</div>

Create, configure, and manage your AI agents (characters).

## List Agents

<div className="api-endpoint">
  <span className="method-badge method-badge-get">GET</span>
  <span className="path">/api/v1/my-agents/characters</span>
</div>

Get all your agents.

### Response

```json
{
  "characters": [
    {
      "id": "char_abc123",
      "name": "Customer Support Bot",
      "bio": ["Expert in customer support"],
      "modelProvider": "openai",
      "settings": {
        "model": "gpt-4o"
      },
      "createdAt": "2024-01-15T10:30:00Z"
    }
  ]
}
```

## Create Agent

<div className="api-endpoint">
  <span className="method-badge method-badge-post">POST</span>
  <span className="path">/api/v1/my-agents/characters</span>
</div>

Create a new agent.

### Request

```json
{
  "name": "My Assistant",
  "bio": ["Expert in customer support", "Friendly and helpful"],
  "modelProvider": "openai",
  "settings": {
    "model": "gpt-4o"
  },
  "system": "You are a helpful customer support assistant.",
  "style": {
    "all": ["professional", "friendly"],
    "chat": ["concise"],
    "post": ["engaging"]
  }
}
```

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | ✓ | Agent name |
| `bio` | array | | Array of bio lines |
| `modelProvider` | string | | `openai`, `anthropic`, etc. |
| `settings.model` | string | | Model to use |
| `system` | string | | System prompt |
| `style` | object | | Communication styles |

### Response

```json
{
  "id": "char_abc123",
  "name": "My Assistant",
  "bio": ["Expert in customer support", "Friendly and helpful"],
  "modelProvider": "openai",
  "createdAt": "2024-01-15T10:30:00Z"
}
```
