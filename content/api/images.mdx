Create images from text descriptions.

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `prompt` | string | ✓ | Description of the image to generate |
| `model` | string | | Model to use. Default: `fal-ai/flux/schnell` |
| `size` | string | | Output size |
| `num_images` | integer | | Number of images (1-4). Default: 1 |
| `negative_prompt` | string | | What to avoid in the image |
| `seed` | integer | | Random seed for reproducibility |
| `guidance_scale` | number | | Prompt adherence (1-20) |

### Available Sizes

| Size | Dimensions |
|------|------------|
| `square` | 512×512 |
| `square_hd` | 1024×1024 |
| `portrait_4_3` | 768×1024 |
| `portrait_16_9` | 576×1024 |
| `landscape_4_3` | 1024×768 |
| `landscape_16_9` | 1024×576 |

### Response

```json
{
  "images": [
    {
      "url": "https://storage.eliza.ai/images/abc123.png",
      "width": 1024,
      "height": 1024,
      "content_type": "image/png"
    }
  ],
  "cost": 0.02,
  "model": "fal-ai/flux/schnell",
  "seed": 12345
}
```

## Available Models

| Model | Quality | Speed | Best For |
|-------|---------|-------|----------|
| `fal-ai/flux/schnell` | High | Fast | General use |
| `fal-ai/flux/dev` | Higher | Medium | Detailed images |
| `fal-ai/flux-pro` | Highest | Slower | Professional |

## Prompt Tips

### Be Specific

```json
{
  "prompt": "A golden retriever puppy playing in autumn leaves, 
             golden hour lighting, shallow depth of field"
}
```

### Use Negative Prompts

```json
{
  "prompt": "Portrait of a woman, natural lighting",
  "negative_prompt": "blurry, distorted, low quality"
}
```
