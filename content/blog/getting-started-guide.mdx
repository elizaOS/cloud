This tutorial will walk you through creating your first AI agent on Eliza Cloud. By the end, you'll have a fully functional agent ready to interact with users.

Before you begin, make sure you have:

- An Eliza Cloud account
- Basic understanding of AI concepts

## Step 1: Create Your Agent

Navigate to the dashboard and click "Create Agent". You'll be prompted to configure the following:

| Setting | Description | Required |
|---------|-------------|----------|
| **Name** | Your agent's display name | Yes |
| **Model** | The AI model to use (GPT-4, Claude, etc.) | Yes |
| **Personality** | Tone and behavior style | No |
| **Knowledge Base** | Custom documents for context | No |
| **Integrations** | Connected services (Discord, Slack) | No |

## Step 2: Configure Capabilities

Next, configure what your agent can do. Here's a complete configuration example:

```javascript
const agentConfig = {
  name: "My First Agent",
  model: "claude-3-sonnet",
  capabilities: ["chat", "search", "summarize"],
  personality: {
    tone: "friendly",
    formality: "casual",
    verbosity: "concise"
  },
  settings: {
    maxTokens: 2048,
    temperature: 0.7,
    streamResponses: true
  }
};
```

### TypeScript Types

For TypeScript users, here are the type definitions:

```typescript
interface AgentConfig {
  name: string;
  model: 'gpt-4' | 'claude-3-opus' | 'claude-3-sonnet';
  capabilities: Capability[];
  personality?: PersonalityConfig;
  settings?: AgentSettings;
}

type Capability =
  | 'chat'
  | 'search'
  | 'summarize'
  | 'code-generation'
  | 'image-analysis';
```

## API Integration

You can also interact with your agent programmatically:

```bash
curl -X POST https://api.eliza.cloud/v1/agents/your-agent-id/chat \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Hello, how are you?",
    "conversationId": "optional-conversation-id"
  }'
```

### Response Format

```json
{
  "id": "msg_abc123",
  "content": "Hello! I'm doing great, thanks for asking. How can I help you today?",
  "role": "assistant",
  "timestamp": "2025-01-10T12:00:00Z",
  "usage": {
    "promptTokens": 12,
    "completionTokens": 18,
    "totalTokens": 30
  }
}
```

## Troubleshooting

### Common Issues

<details>
<summary><strong>Agent not responding</strong></summary>

Check the following:
1. Verify your API key is valid
2. Ensure you have available credits
3. Check the agent status in dashboard

</details>

<details>
<summary><strong>Slow response times</strong></summary>

Try these solutions:
1. Switch to a faster model (e.g., GPT-3.5)
2. Reduce max token limit
3. Enable streaming responses

</details>

<details>
<summary><strong>Unexpected responses</strong></summary>

Consider:
1. Adjusting the temperature setting
2. Refining the personality prompt
3. Adding more context to the knowledge base

</details>
