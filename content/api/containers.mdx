# Containers

<div className="status-badge status-stable">Stable</div>

Deploy and manage containerized applications.

## List Containers

<div className="api-endpoint">
  <span className="method-badge method-badge-get">GET</span>
  <span className="path">/api/v1/containers</span>
</div>

Get all your containers.

### Response

```json
{
  "containers": [
    {
      "id": "cnt_abc123",
      "name": "my-agent-runtime",
      "status": "running",
      "url": "https://my-agent.elizacloud.ai",
      "createdAt": "2024-01-15T10:30:00Z"
    }
  ]
}
```

## Create Container

<div className="api-endpoint">
  <span className="method-badge method-badge-post">POST</span>
  <span className="path">/api/v1/containers</span>
</div>

Deploy a new container.

### Request

```json
{
  "name": "my-agent-runtime",
  "characterId": "char_abc123",
  "environment": {
    "NODE_ENV": "production"
  }
}
```

### Response

```json
{
  "id": "cnt_abc123",
  "name": "my-agent-runtime",
  "status": "deploying",
  "url": "https://my-agent.elizacloud.ai"
}
```

## Container Logs

<div className="api-endpoint">
  <span className="method-badge method-badge-get">GET</span>
  <span className="path">/api/v1/containers/{'{id}'}/logs</span>
</div>

Get container logs.

### Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `tail` | integer | Number of lines (default: 100) |
| `since` | string | Start time (ISO 8601) |

### Response

```json
{
  "logs": [
    {
      "timestamp": "2024-01-15T10:30:00Z",
      "message": "Server started on port 3000"
    }
  ]
}
```
