## Global Lessons

- When a review comment requests test coverage, the fix must include actual test files with executable test cases.
- When fixing a two-endpoint flow, address both the producer and consumer endpoints.
- When converting service calls to use transactions, verify that each service method actually accepts and uses the transaction parameter.
- When fixing code duplication, update all call sites to import from the shared module.
